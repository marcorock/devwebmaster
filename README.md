## :checkered_flag: Iniciando ##

```bash
# Clone this project
$ git clone https://github.com/marcorock/devwebmaster

# Access
$ cd devwebmaster

# Install dependencies
$ yarn

# Run the project
$ yarn start

# The server will initialize in the <http://localhost:3000>
```
